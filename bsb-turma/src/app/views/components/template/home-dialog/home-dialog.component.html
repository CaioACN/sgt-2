Usando o SGT

Primeiros passos:

Para começar a gerenciar as tarefas pelos sistema é fundamental que o cadastro do Setor responsável seja realizado anteriormente, assim como o cadastro do Tipo de Tarefa.

Preenchimento das informações:

As informações de cada campo seguem um padrão, exemplificado em cada campo. Caso os campos não estejam preenchidos de acordo com os exemplos o campo não é salvo no sistema.

Campo Comentários:

O Campo comentário é um campo dependente do Campo Tarefa, deste forma, este só pode ser preenchido após o cadastramento de uma tarefa.
